# BookIt API

A RESTful booking platform built with FastAPI where users can browse services, make bookings, and leave reviews.

## Tech Stack

* FastAPI
* Python
* SQLAlchemy
* Alembic

## Features

* User authentication
* Service management
* Booking system
* Reviews

## Setup

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Set up `.env` file
4. Run migrations: `alembic upgrade head`
5. Start server: `uvicorn main:app --reload`
